# Changelog

## 2.0.0 - 2026-07-20

- Replaced the SSH-only server domain with resource, profile, request, grant, session, event and artifact entities.
- Added SSH and Playwright/Chromium web providers with resource-type dispatch.
- Added CDP screencasting, constrained session-bound input, isolated contexts, heartbeat/timeouts and unified termination.
- Added web authentication modes, worker-only secret injection and sensitive event redaction.
- Added DNS/SSRF/rebinding protection, domain rules, recording/trace checksums and authenticated MFA-gated replay.
- Replaced server APIs and UI with resource workflows; removed deprecated server endpoints and compatibility models.
- Added explicit schema-v2 validation, backup/reset commands, SSH/web demo resources and persistent deployment storage.
